# SCT_WD_1 — Responsive Landing Page (Task 01)

This is a simple responsive landing page built for SkillCraft Task 01.

## Features
- Fixed navigation visible on all pages
- Navigation changes style (background, shadow) on scroll
- Hover and focus effects on nav links
- Mobile hamburger menu
- Smooth scrolling and active link highlighting
- Accessible (ARIA attributes + keyboard support)

## How to run locally
1. Clone or download this folder.
2. Open `index.html` in a modern browser.
